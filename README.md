# 六种数字调制信号识别的MATLAB程序

## 资源描述

本仓库提供了一个基于高阶累积量的六种数字调制信号识别的MATLAB程序。该程序能够识别以下六种数字调制信号：

1. 2ASK
2. 4ASK
3. 2FSK
4. 4FSK
5. 2PSK
6. 4PSK

## 程序功能

该MATLAB程序利用高阶累积量的方法，对输入的数字调制信号进行识别。程序能够准确区分2ASK、4ASK、2FSK、4FSK、2PSK和4PSK六种调制信号，适用于通信信号处理和调制识别的相关研究。

## 使用方法

1. 下载本仓库中的MATLAB程序文件。

3. 根据程序提示，输入待识别的数字调制信号数据。
4. 运行程序，获取识别结果。
